# Computer-Science-NetConnect
You will design, implement, test, analyze, document, and present a fully functioning LAN-based collaborative file sharing system using the school network. The project directly supports the learning outcomes from IB Computer Science Topic 3.1 Networks (2014 curriculum).

Download from git-scm.com and install it:
Once installed, open your terminal (Mac) or Git Bash (Windows) and run:
bashgit config --global user.name "Your Full Name"
git config --global user.email "your@email.com"

Clone the Repository:
git clone https://github.com/OWNER/cs-group-project.git
cd cs-group-project

Every Time You Work on the Project 
Follow these steps every single time, in this order:
Step 1 — Pull the latest changes first
bashgit pull

Always do this before you start. It prevents most conflicts.

Step 2 — Make your changes
Edit whichever files you've been assigned. Try to stick to your own section to avoid conflicts with others.

Step 3 — Check what you've changed
bashgit status

Step 4 — Stage your changes
bashgit add .
